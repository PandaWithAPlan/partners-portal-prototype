Часто задаваемые вопросы

Для активации лицензии перейдите в раздел "Лицензии", нажмите кнопку "Активировать" и введите полученный лицензионный ключ. Система автоматически проверит ключ и активирует продукт.

На странице входа нажмите "Забыли пароль?". Введите email, указанный при регистрации, и следуйте инструкциям в письме для восстановления доступа.

Минимальные требования: Windows 10/macOS 10.15, 8GB RAM, современный браузер (Chrome, Firefox, Safari, Edge). Для корпоративных решений рекомендуется 16GB RAM.

Обновления устанавливаются автоматически. Для ручного обновления перейдите в "Настройки" → "Обновления" и нажмите "Проверить обновления". Следуйте инструкциям мастера обновления.

Документация по API доступна в разделе "Разработчикам" → "API документация". Там вы найдете описание всех методов, примеры запросов и ответов, а также SDK для популярных языков программирования.
